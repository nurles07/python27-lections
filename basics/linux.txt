cd test - переходим в папку test (если это папка и она существует)
к любой команде можно дописать --help и выйдет описание как использывать эту команду


cd ..    - переход в родительску папку 
cd .     -переход в эту же папку (мы не куда не перейдем )
cd        - переход в домашнюю папку (~)
cd~       - переход в домашнюю папку (~)
pwd      - показывает абсолютный путь до текущей папки 
mkdir test     - создание новой папки test 
mkdir test/new_dir     -создание новой папки new_dir в уже существующей папке test 

touch test.txt        -создание файла test.txt 


rm test.txt      -удаление файла test.txt
rm -rf test      -это удаление папки test со всем ее содержимым 
cat test.txt       -просмотр содежимого файла test.txt 
ls test        -просмотр всех файлов и папок внутри папки test 
ls -a test         просмотр всех файлов и папок внутри папки test включая скрытые 

ls -l            просмотр всех файлов и папок более с подробной информацией
nano test.txt       - открыть файл в редакторе nano
чтобы сохранить (ctrl s)
чтобы выйти (ctrl x)


!!!НИКОГДА НЕ ИСПОЛЬЗЫВАТЬ   sudo rm -rf  формат чистка
